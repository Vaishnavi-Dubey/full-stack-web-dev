CSS Relative Positioning

Normal Positioning (static by default):
This box is in the normal document flow. No offset applied.
Relative Positioning:
This element is positioned relative to its normal position. The top: 20px; and left: 30px; move it down and right respectively.
Important: The space the element originally occupied remains reserved, so other elements behave as if it is still in the original position.
Next Element:
This box appears as if the relative box had not moved, because relative positioning offsets the element visually but does not affect the normal document flow.
