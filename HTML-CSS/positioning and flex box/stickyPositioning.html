Sticky Header (sticks when you scroll down)

Scroll down to see how the header sticks to the top of the viewport when it reaches it.

Sticky positioning is a hybrid of relative and fixed positioning. The element behaves like relative until it reaches a defined threshold (top: 0 here), then it becomes fixed.

This is useful for navigation menus or section headers that you want visible as you scroll but only after they reach the top.
