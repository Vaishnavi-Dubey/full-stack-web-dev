Fixed Positioning Example

Scroll down this page to see that the header remains fixed at the top of the viewport.

Fixed positioning means the element is positioned relative to the browser window (viewport), so it does not move when you scroll the page.

This is useful for navigation bars, headers, footers, or any UI elements you want visible at all times.
